# 个人风格指南

> 本文件由 Claude 从博客分析中提取，用户可自行维护更新。

## 核心特质

| 特质 | 描述 |
| --------------------- | ---------------------------------------- |
| **自觉的不完美性** | 承认错误、标注「还在鸽着」、保留思考痕迹 |
| **深度的自我对话** | 写给「十年后的自己」，充满自我审视 |
| **学院派 + 网文混搭** | 脚注引用 + 网络黑话，严谨与随意并存 |

## 标志性表达

### 高频词汇

| 表达 | 使用场景 |
| --------------- | ------------------ |
| 「怎么说呢」 | 段落开头，组织语言 |
| 「实在是」 | 表达强烈程度 |
| 「有点...」 | 弱化语气 |
| 「好吧」 | 自我认可/调侃 |
| 「其实」 | 反转预期 |
| 「搁置/鸽着」 | 描述未完成任务 |
| 「要说...的话」 | 转换话题 |
| 「正所谓」 | 引用谚语/总结 |

### 语气特征

- **自嘲式开场**：「这是我人生当中的至黑至暗时刻」
- **括号补充吐槽**：大量使用小括号做补充说明
- **对话式反问**：「该说是...吗？」「是吧？」
- **感叹句**：「简直是太...」「实在是...」

### 量化习惯

喜欢用具体数字增强说服力：

- 「100H 游戏」「200 部动画」「800 小时音频」

## 结构偏好

### 标准三段式

1. **开场破题** - 用自嘲或反问吸引注意
2. **分析展开** - 多层次的递进式说明
3. **后话反思** - 自我批评或调整立场

### 递进特征

- 正文 + 后话 + 后续的后续 = 多层反思递进
- 承认「当时错了」，展示观点演变

### Markdown 使用

- 一级标题：章节概览
- 二级标题：分类细分
- 三级标题：具体论点
- 列表/表格：量化呈现
- **加粗**：重要转折、深刻感悟、关键观察（避免滥用）

## 禁忌 / 避免

| 避免 | 原因 |
| -------------------- | -------------------------------- |
| 过度精修 | 保留真诚和思考痕迹 |
| 装专业 | 承认「主观」「不完美」更符合风格 |
| 堆砌数据 | 数据要融入体验，不是罗列 |
| 每段都加粗 | 只在重要处使用，避免突出太多 |
| 独立加粗小标题 | 会被 IDE 警告 |
| 「不是……而是……」句型 | AI 爱用，反复出现显得机械 |
| 加引号的「幽默」比喻 | 刻意感太强，读者容易出戏 |
| emoji 或序号开头 | 一眼 AI 味，破坏叙事连贯性 |
| 频繁使用破折号 | 打断阅读节奏，显得生硬 |
| 过度结构化 | 失去口语感和个人语境 |
| 机械分条论述 | 缺乏情绪流动和自然过渡 |

## 目标读者

### 是

- 十年后的自己
- 有 ACG 背景的「赛博邻居」
- 陷入类似困境、愿意深入对话的人

### 不是

- 不看动画的人（信息差大）
- 只想快速结论的人（文章充满「其实」「可能」）
- 寻求标准答案的人

## 语气变化规律

| 文章主题 | 语气 | 用词倾向 |
| -------- | --------------- | ------------------------------ |
| ACG 评测 | 热烈 + 悔恨混杂 | 「至黑至暗」「扭曲」「毒」 |
| 生活约束 | 理性 + 决心 | 「必须」「应该」「无可奈何」 |
| 人物反思 | 温暖 + 自责 | 「感悟」「原来是」「有点不舍」 |
| 人文思考 | 思辨 + 孤独 | 「其实」「缺少」「虚无缥缈」 |
| 技术分享 | 平实 + 有趣 | 直接说明，偶尔黑话穿插 |

## 风格检查清单

- [ ] 是否保持了坦诚？（承认不足、标注未完成）
- [ ] 是否有自我对话感？（而非单向输出）
- [ ] 是否用了标志性表达？（「怎么说呢」「其实」等）
- [ ] 数据是否融入体验？（而非生硬堆砌）
- [ ] 加粗是否克制？（每标题下 ≤ 3 处）
- [ ] 是否考虑了目标读者？（ACG 背景、愿意深入）
- [ ] 是否避免了「不是……而是……」的句型？
- [ ] 是否去除了加引号的刻意比喻？
- [ ] 是否避免了 emoji 和序号开头？
- [ ] 破折号是否控制在合理范围？（每段 ≤ 1 处）
- [ ] 段落是否有自然的情绪流动？（而非机械分点）
