# 写作框架参考

详细的启发问题和框架参考，用于各类文章类型。

## 游记详细框架

### 时间线组织

```text
Day 1: 到达
  - 第一印象
  - 安顿下来
  - 初次探索

Day 2-N: 主要行程
  - 每天的亮点
  - 意外发现
  - 美食体验

Last Day: 离开
  - 最后的收获
  - 依依不舍
  - 下次再来的计划
```

### 主题组织

```text
美食篇
  - 当地特色
  - 意外发现
  - 踩雷经历

景点篇
  - 必去打卡
  - 小众推荐
  - 不值得去

人文篇
  - 当地人印象
  - 文化冲击
  - 语言趣事
```

### 感官维度

| 维度 | 启发问题 |
|------|----------|
| 视觉 | 最美的一幕是什么？有什么颜色让你印象深刻？ |
| 听觉 | 当地有什么特别的声音？音乐、方言、自然声？ |
| 嗅觉 | 有什么气味让你一闻就想起这个地方？ |
| 味觉 | 最难忘的一道菜是什么？有什么意外的味道？ |
| 触觉 | 天气、温度、材质？有什么触感让你印象深刻？ |

## TIL 详细框架

### 问题描述模板

````markdown
## 背景

遇到了 [什么问题]，表现为 [具体症状]。

环境：
- OS: macOS 14.x
- Node: v20.x
- 其他相关版本

## 尝试过的方法

1. [方法 1]：[结果]
2. [方法 2]：[结果]
3. [方法 3]：[结果]

## 解决方案

[具体解决步骤]

## 关键代码

```code
// 核心代码
```

## 收获

- [收获 1]
- [收获 2]

````

### 常见 TIL 类型

| 类型 | 特点 | 重点 |
|------|------|------|
| Bug 修复 | 从错误到正确 | 错误原因、调试过程 |
| 新技术学习 | 从零到入门 | 核心概念、实践例子 |
| 性能优化 | 从慢到快 | 对比数据、关键改动 |
| 配置踩坑 | 从不工作到工作 | 正确配置、易错点 |

## 通用文章详细框架

### 观点类文章

```text
1. Hook（开头吸引）
   - 反直觉观点
   - 个人故事
   - 数据震撼

2. 背景（为什么重要）
   - 问题现状
   - 常见误解
   - 你的视角

3. 主体（论证）
   - 论点 1 + 证据
   - 论点 2 + 证据
   - 论点 3 + 证据

4. 反驳（可选）
   - 预期的反对意见
   - 你的回应

5. 结尾（行动召唤）
   - 总结核心
   - 下一步建议
```

### 叙事类文章

```text
1. 情境设定
   - 时间、地点、人物
   - 氛围营造

2. 冲突/转折
   - 发生了什么
   - 内心变化

3. 高潮
   - 最关键的时刻
   - 最深的感受

4. 结局
   - 事情如何收场
   - 带来了什么改变
```

## 周记 / 月记扩展

在日记框架基础上增加的问题：

| 维度 | 问题 |
|------|------|
| **Review** | 这周/月的整体状态如何？ |
| **Highlights** | 最值得记录的 3 件事？ |
| **Lowlights** | 最想吐槽的 3 件事？ |
| **Next** | 下周/月想做什么？ |

## 启发技巧

### 追问模式

当用户回答比较简短时：

```text
用户: 「今天去了公园」
Claude: 「公园里有什么让你印象深刻的吗？或者有什么意外的发现？」
```

### 具象化

当用户回答比较抽象时：

```text
用户: 「感觉很开心」
Claude: 「是什么让你开心？有没有一个具体的瞬间？」
```

### 对比

帮助用户发现新角度：

```text
Claude: 「这次和上次旅行相比，有什么不同的感受吗？」
```
