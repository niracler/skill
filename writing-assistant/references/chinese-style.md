# 中文写作规范

基于余光中《怎样改进英式中文》整理。

## 常见问题与纠正

### 1. 抽象名词滥用

中文不习惯用抽象名词做主语。

| 问题写法             | 改进写法                     |
| -------------------- | ---------------------------- |
| 收入减少改变生活方式 | 他因为收入减少而改变生活方式 |
| 这个决定影响了结果   | 他这样决定，结果就不同了     |
| 效率的提高带来了收益 | 效率提高后，收益也跟着增加   |

### 2. 冗赘句式

避免把简单表达复杂化。

| 冗赘          | 简洁                     |
| ------------- | ------------------------ |
| 基于这个原因  | 因此                     |
| 关于这个问题  | 这个问题                 |
| 对于...而言   | 对...来说 / 直接省略     |
| 在...的情况下 | ...时                    |
| 具有...的特点 | ...的特点是 / 直接说特点 |
| 进行讨论      | 讨论                     |
| 作出决定      | 决定                     |
| 给予帮助      | 帮助                     |

### 3. 动词弱化

简单动词被「抽象名词 + 泛动词」替代。

| 弱化写法 | 直接写法 |
| -------- | -------- |
| 作出贡献 | 贡献     |
| 进行研究 | 研究     |
| 给予支持 | 支持     |
| 做出努力 | 努力     |
| 进行分析 | 分析     |
| 加以考虑 | 考虑     |

### 4. 介词滥用

「有关」「关于」「对于」等介词过度使用。

| 滥用         | 改进              |
| ------------ | ----------------- |
| 有关这件事   | 这件事            |
| 关于他的问题 | 他的问题          |
| 对于我来说   | 我觉得 / 在我看来 |
| 针对这个情况 | 这个情况          |

### 5. 连接词堆砌

「和」「与」「及」「以及」过多。

| 堆砌             | 改进             |
| ---------------- | ---------------- |
| 他和我和他们     | 他、我、他们     |
| A 以及 B 以及 C  | A、B、C          |
| 苹果与香蕉与橘子 | 苹果、香蕉、橘子 |

### 6. 复数语尾滥用

| 滥用   | 改进             |
| ------ | ---------------- |
| 人们   | 人人、大家、众人 |
| 学生们 | 学生、诸生       |
| 朋友们 | 朋友、诸位       |

### 7. 被动语态滥用

中文偏好主动语态。

| 被动               | 主动             |
| ------------------ | ---------------- |
| 这本书被很多人喜欢 | 很多人喜欢这本书 |
| 问题被解决了       | 问题解决了       |
| 他被认为是专家     | 大家认为他是专家 |

## 成语活用

适当使用成语，保持文化传承：

| 白话     | 成语                 |
| -------- | -------------------- |
| 总的来说 | 总而言之、一言以蔽之 |
| 最后     | 归根结底、说到底     |
| 很多     | 不胜枚举、数不胜数   |
| 到处都是 | 比比皆是、俯拾皆是   |
| 非常重要 | 举足轻重、至关重要   |

## 审校清单

审校时逐项检查：

- [ ] 抽象名词是否做了主语？
- [ ] 有无冗赘句式可简化？
- [ ] 动词是否被弱化？
- [ ] 介词是否过多？
- [ ] 连接词是否堆砌？
- [ ] 复数语尾是否必要？
- [ ] 被动语态是否可改主动？
- [ ] 有无适合使用成语的地方？
- [ ] 有无错别字、歧义？
